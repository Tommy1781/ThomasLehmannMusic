# Thomas Lehmann Music – GitHub Pages

Dies ist eine minimalistische, zweisprachige (DE/EN) Musiker-Website im Stil **Cinematic Harmony**.

## So ersetzt du die YouTube-IDs
In `index.html` findest du Stellen wie:

```html
<iframe src="https://www.youtube.com/embed/VIDEO_ID_1" ...></iframe>
```

- Ersetze `VIDEO_ID_1` usw. mit deinen echten YouTube-IDs (z. B. `dQw4w9WgXcQ`).
- Du kannst beliebig viele Karten kopieren (Block `<article class="card">...</article>`).

## Hochladen
1. Gehe in dein Repository → **Add file → Upload files**
2. Lade alle Dateien aus diesem Ordner hoch (am einfachsten: die ZIP entpacken und alles markieren).
3. Speichern (Commit).
4. Deine Seite aktualisiert sich in 30–60 Sekunden.
